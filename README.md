# Project-2
1. Introduction

The Bus Reservation System is a console-based application developed using C++ that helps manage bus ticket bookings efficiently. This system allows users to reserve seats, view available buses, cancel reservations, and check booking details. It reduces manual work and ensures accurate record-keeping.

2. Objective

The main objectives of this project are:

To provide an easy interface for booking bus tickets
To maintain records of passengers and reservations
To reduce errors in manual booking systems
To demonstrate the use of C++ concepts like classes, objects, file handling, and functions
3. Features
Display available buses
Seat reservation system
Ticket cancellation
View reservation details
Store data using file handling
User-friendly menu-driven interface
